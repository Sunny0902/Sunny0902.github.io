2.数组

1.Js有哪些数据类型：

(1) 基本数据类型 Number String Boolean Null Undefined

(2) 引用数据类型 Function Object

2.数组：由若干个元素(可以是不同的数据类型)组成的集合

3.使用格式：

(1) 数组的声明：

var arr = [1,2,3]; //声明数组的时候就赋值

var arr = new Array(); //声明一个空数组

(2)数组赋值:

arr[0] = “aa”;

arr[1] = “bb”;

注：获取数组元素：仍然通过下标来获取，但是千万注意下标不要越界。

​ 数组的下标是从0开始统计的，比如数组中一个有10个元素，下标只能用到9

(3)数组的长度：arr.length

(4) 数组的遍历：

for (var i = 0; i < scores.length; i++) {

console.log(scores[i]);

}
