1. JS简介

JS是一门弱类型的基于对象的解释型的动态的脚本语言。

脚本：理解为文本，可以把js代码当成普通文本嵌入到html网页中执行。

解释型：编程语言可以分为两种，一种是编译型，一种是解释型。

弱类型：在js中变量的声明都用var关键字，也就是说我们在声明变量的时候不去关心它的数据类型，变量的数据类型究竟是什么是由变量的值推导出来的。

1.2 变量

变量是计算机内存中存储数据的标识符，根据变量名称可以获取到内存中存储的数据

声明变量的规则：

  1. 变量需要是数字、字母、下滑线、美元符组成
  2. 变量不能以数字开头
  3. 变量不能与关键字重名 if else switch for while do var let const
  4. 变量严格区分大小写
  5. 当有多个英文单词的时候遵循驼峰命名法(后面每个单词的首字母大写 / 或者用_连接单词)

1.3 JS中的数据类型

基本数据类型：Number String Boolean Undefined Null

引用数据类型：Function Object

1.3.1 Number类型

console.log(Number.MAX_VALUE) //Number类型的最大值

console.log(Number.MIN_VALUE) //Number类型的最小值,无限接近于0,不包含负数

1.3.2 String类型

  1. js中单引号和双引号都可以表示字符串 ,单引号中可以套双引号字符串，双引号中可以套单引号字符串，但是双引号不可以有双引号字符串，单引号中不可以有单引号字符串

  3. 字符串的length方法：求字符串的长度

    var str8 = “前端程序员”;

    console.log(str8.length);

  4. 字符串的运算

    字符串和其他任何东西相加得到字符

    数值字符串在和其他数值运算的时候，会先将数值字符串转换为数值然后再运算

1.3.3 Boolean类型

Boolean类型的值只有两个，一个true, 一个flase

1.3.4 Null和Undefined

  1. Undefined类型只有一个值，即undefined。undefined表示一个声明了没有赋值的变量，变量只声明的时候值默认是undefined

  2. Null类型也只有一个值，即null。null表示一个空，null值表示一个空对象指针。变量的值如果想为null，必须手动设置
